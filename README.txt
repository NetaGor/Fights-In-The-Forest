Welcome to Fights in the Forest!
A multiplayer strategy combat game inspired by D&D, bringing tabletop RPG combat to mobile devices with real-time multiplayer functionality.

Prerequisites
Client (Android):
	Android 11.0+ (API level 31+)
	Internet connection

Server:
	Python 3.9+
	Access to Firebase database
	Required python packages (see requirements.txt)
	

Missing files:
public/private key txt files for the encryption
firebase SDK
